Noltron000's Downtime Portfolio

This webpage is dedicated to the projects which I work on during my downtime. Most of these projects are related games, but not all. Some of my projects actually produce relevent and usable results, while others are merely experiences that aren't particularly useful for anyone except me.
Since this website is somewhat lengthy, I suggest first looking at my Dice Roller page, which I set up specifically for the Holberton application.
