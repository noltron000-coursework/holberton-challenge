Character Sheet Printouts

Optimizing Player's Papers

I'm an avid Game Master for D&D. I love telling stories and having players experinece the worlds and scenarios that I create, but the mechanics can get confusing. Sometimes that confusion can be avoided with proper printouts. So, I took to a task: improve on player character printouts. I mostly use photoshop on school computers to complete these.

Modularity

One thing I aim for is creating modules in my printouts. I want to be able to move things around, maybe add them or remove them, see what works best. I want players to be able to add their own stat-bars as well. While my printout is not complete, I hope that you can see the parts that I improved upon. If not, I suggest comparing the health box for each character sheet. I tried to make mine more conceptually concrete for new players.
